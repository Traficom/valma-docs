# Asennus

Ohjeessa kuvataan Bentleyn OpenPaths EMME -ohjelmiston sekä VALMA-käyttöliittymän asennus.

## EMME-ohjelmiston asennus

VALMA UI -käyttöliittymän käyttö edellyttää EMME‑ohjelmistoa, tunnuksia ja lisenssiä.

Voit hankkia nämä [Bentleyn nettisivuilta](https://www.bentley.com/software/openpaths/).
- Tarkemmat ohjeet ohjelmiston asentamiseen löydät [täältä](https://docs.bentley.com/LiveContent/web/OpenPaths%20Help-v1/en/GUID-91E9693E-4255-48EE-B011-27E105B8CD57.html).

## VALMA-käyttöliittymän asennus

Käyttöliittymän asennustiedosto ladataan GitHubista Liikenne- ja viestintävirasto Traficomin lem-ui-repositorion [Releases-sivulta](https://github.com/Traficom/lem-ui/releases). Asennustiedosto on nimeltään `VALMA-x.y.z.Setup.exe`, missä `x.y.z` on käyttöliittymän versionumero (esimerkiksi `_VALMA-1.0.13.Setup.exe_`). Käyttöliittymällä voidaan käyttää mallijärjestelmän eri versioita, mutta suosittelemme käyttämään uusinta versiota. Uusin versio sijaitsee aina ylimpänä _Releases_-sivulla.
